# dotfiles

Personal Mac setup automation. See `docs/superpowers/specs/2026-06-25-mac-setup-design.md` for the design.

## Bootstrap a fresh Mac

```bash
curl -fsSL https://raw.githubusercontent.com/<GH_USER>/dotfiles/main/bootstrap.sh | bash
```

(Replace `<GH_USER>` with your GitHub username before publishing.)

Full bootstrap docs and 1Password vault prerequisites: see `docs/1password-setup.md` (added in a later task).
